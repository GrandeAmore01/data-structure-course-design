# 使用指南

## 运行程序

### 方法一：使用批处理脚本

双击运行 `run.bat` 文件，脚本会自动检查环境并启动程序。

### 方法二：手动运行

1. 确保已安装 Java 11+ 和 Maven
2. 在项目根目录打开命令行
3. 执行：`mvn javafx:run`

### 方法三：编译后运行

```bash
mvn clean package
java --module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml -cp target/classes com.datastruct.visualizer.DataStructureVisualizerApp
```

## 功能使用说明

### 图数据结构

#### 创建图

1. 选择存储类型：
   - **邻接矩阵**：适合稠密图，支持快速边查询
   - **邻接表**：适合稀疏图，节省空间
2. 输入顶点数量（1-20）
3. 点击"创建"按钮

#### 添加/删除边

1. 在"起点"和"终点"输入框中输入顶点索引（从 0 开始）
2. 设置边的权重（默认 1.0）
3. 点击"添加边"或"删除边"

#### 运行图算法

1. 选择算法类型：
   - **深度优先搜索（DFS）**：递归遍历，展示回溯过程
   - **广度优先搜索（BFS）**：层次遍历，使用队列
   - **最小生成树**：构建最小权重连通子图
2. 设置起始顶点索引
3. 点击"运行算法"观看动画演示

### 排序算法

#### 设置数组

1. 在输入框中输入数组元素，用逗号或空格分隔
   - 示例：`64, 34, 25, 12, 22, 11, 90`
2. 点击"设置数组"按钮

#### 选择排序算法

- **直接插入排序**：从第二个元素开始，逐个插入到已排序序列
- **简单选择排序**：每次选择最小元素放到已排序序列末尾
- **快速排序**：分治策略，选择基准元素进行分割

#### 控制动画

- **开始排序**：开始/重新开始排序动画
- **暂停/继续**：暂停或继续动画播放
- **重置**：回到初始状态
- **速度滑块**：调整动画播放速度（100-2000ms）

### 可视化说明

#### 图的可视化

- **顶点**：蓝色圆圈，带有顶点标签
- **边**：黑色直线，有向图显示箭头
- **权重**：显示在边的中点
- **高亮**：红色表示当前操作的顶点/边

#### 排序的可视化

- **柱状图**：高度表示数值大小
- **颜色编码**：
  - 蓝色：默认状态
  - 黄色：比较操作
  - 红色：交换操作
  - 绿色：高亮显示
  - 浅绿色：排序完成

### 高级功能

#### 数据保存与加载

- 点击"文件"菜单中的"保存"可将当前数据结构保存为 JSON 文件
- 点击"加载"可从文件恢复之前的数据结构

#### 快捷键

- `F5`：重新运行当前算法
- `Space`：暂停/继续动画
- `Esc`：停止当前动画

## 故障排除

### 常见问题

1. **程序无法启动**

   - 检查 Java 版本（需要 11+）
   - 检查 JavaFX 是否正确安装
   - 查看控制台错误信息

2. **动画卡顿**

   - 降低动画速度
   - 减少数据规模
   - 关闭其他占用资源的程序

3. **图显示异常**

   - 检查顶点索引是否在有效范围内
   - 确保边的权重为有效数值
   - 重新创建图结构

4. **排序结果错误**
   - 检查输入数组格式
   - 确保数组元素为整数
   - 查看算法信息面板的详细说明

### 获取帮助

- 查看控制台输出的错误信息
- 参考项目 README.md 文件
- 检查日志文件（如果生成）
